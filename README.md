# HALE GitHub Pages / Black Ops Upgrade

이 버전은 기존 GitHub Pages 구조를 유지한 채, 전체 디자인을 프리미엄 블랙-옵스 / 시크릿 인텔리전스 포털 스타일로 업그레이드한 구성입니다.

구성:
- `index.html` : 메인 페이지 구조
- `assets/css/main.css` : 전체 스타일 시스템
- `assets/js/vendor/instanced-mouse-effect.bundle.js` : WebGL 타일 애니메이션 번들
- `assets/js/app/main.js` : 인터랙션, 시계, 스크롤, 카운터, 페이지 보조 애니메이션

포함된 주요 업그레이드:
- 더 강한 HALE 타이포그래피와 레이어드 타이틀
- 고급 대기업 사이트 수준의 상단 바 / 네비게이션 / 상태 인디케이터
- 프로필 / 스토리 / 아카이브 섹션 추가
- 스크롤 리빌, 카운터, 런던 시계, 포인터 기반 광원/패럴랙스
- 상대 경로 기반이라 GitHub Pages에서 그대로 동작

배포 방법:
1. 이 폴더 내용을 GitHub 저장소 루트 또는 `docs/` 폴더에 업로드
2. GitHub 저장소의 Settings → Pages 에서 배포 경로 선택
3. 배포 후 생성된 링크로 접속
